ChatGPTApp
=============

https://platform.openai.com/docs/api-reference/completions API를 참고하여 만든 ChatGPT 질문 앱

## Tuist

이 프로젝트는 `Tuist`를 사용하여 Xcode 프로젝트를 생성합니다.  
`tuist generate` 명령으로 `ChatGPTApp.xcodeproj`를 다시 만들 수 있습니다.

## OpenAI 토큰 관련 안내

레포지토리에는 **동작하지 않는 임의의 ChatGPT(OpenAI) 토큰 값**이 들어가 있습니다.  
실제로 앱을 실행하려면 OpenAI 대시보드에서 발급받은 **본인 계정의 API 키**를 설정 파일/환경설정에 따로 넣어줘야 합니다.
